# Raymarching with SDF

Experiments with raymarching and signed distance functions (SDF) in WebGL.

## Tech

- WebGL2 via [twgl.js](www.twgljs.org)
- GLSL shaders
- JavaScript (ES6 modules)
- Vite for dev server and letting us import GLSL files
